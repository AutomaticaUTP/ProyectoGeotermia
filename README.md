# ProyectoGeotermia

Repositorio oficial para el desarrollo de códigos, experimentos y herramientas computacionales asociadas al proyecto de geotermia. Este repositorio integra diferentes componentes del flujo de trabajo del proyecto, desde la estimación del gradiente geotérmico y la simulación de sistemas de generación de energía, hasta el uso de modelos avanzados de aprendizaje automático y técnicas de procesamiento de imágenes.

## Propósito del repositorio

El objetivo de este repositorio es consolidar, organizar y documentar el código fuente empleado en las distintas etapas del proyecto, facilitando:

- la reproducibilidad de experimentos,
- la trazabilidad del desarrollo metodológico,
- la colaboración entre investigadores,
- y la escalabilidad del proyecto hacia nuevas aplicaciones.

## Estructura general del proyecto

El repositorio se encuentra organizado en tres grandes etapas, cada una asociada a un componente clave del proyecto de investigación.
