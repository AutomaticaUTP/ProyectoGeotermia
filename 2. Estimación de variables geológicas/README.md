# Estimación de variables geológicas y técnicas

<p align = 'justify'>En esta carpeta del repositorio se encuentra contenida los modelos para estimar el gradiente geotérmico, las caídas de presión en los pozos y los perfiles de temperatura del fluido.

---

## Vertientes de la estimación

- En la carpeta de estimación del gradiente geotérmico, se encuentran contenidos los códigos de preprocesamiento y de los modelos desarrollados para la estimación del gradiente geotérmico a partir de la integración de datos tabularesstrong> e imágenes satelitales multiespectrales.

- En la carpeta de parámetros del pozo, se encuentran dos modelos: el primero es la implementación de un modelo térmico unidimensional de pozo vertical tipo Ramey/line-source para estimar perfiles de temperatura del fluido y temperaturas de salida en función del tiempo. El segundo implementa un modelo hidráulico simplificado para estimar la caída de presión en pozos verticales bajo flujo líquido–líquido (aceite–agua), utilizando una aproximación homogénea y propiedades
constantes.
