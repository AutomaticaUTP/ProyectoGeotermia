# Modelado y simulación de planta de Ciclo Rankine Orgánico (ORC)

Este repositorio contiene las implementaciones computacionales del modelo físico de una planta ORC

## 📂 Estructura del Proyecto

### 1\. Modelo estacionario de la planta ORC

Esta carpeta contiene el **Simulador Dinámico Transitorio** desarrollado en Python (PyTorch + Streamlit).

  * **Enfoque:** Parámetros concentrados para el análisis de estados transitorios e inercia del sistema.
  * **Contenido:** Código fuente del tablero interactivo, requisitos de entorno y documentación detallada del modelo matemático de ecuaciones diferenciales acopladas.
